# 数据结构习题集（选择题练习系统）

> 可用于 **淮安大学** 学习复习使用

## 教材信息

- **书名**：《数据结构（C语言版 第3版）》
- **作者**：李冬梅、严蔚敏、吴伟民 编著
- **出版社**：人民邮电出版社 / 中国工信出版集团
- **性质**：教材配套教学资源 — 数据结构第3版习题集

## 包含内容

| 文件 | 说明 |
|------|------|
| `数据结构习题集_题目与解析.html` | 全部章节题目 + 详细解析，可离线浏览 |
| `数据结构选择题练习.html` | 交互式刷题系统，支持错题集、章节筛选、重点练习 |

## 使用方式

1. 下载仓库后，直接用浏览器打开 `.html` 文件即可
2. 选择题练习系统支持：
   - 按章节筛选题目
   - 错题自动收录、反复练习
   - 重点题目标记（根据老师考试范围）
   - 手动上一题/下一题导航

## 考试范围（第7、8章不考）

- 第1章 绪论（基础）
- 第2章 线性表（重点）
- 第3章 栈和队列（重点）
- 第4章 串、数组和广义表（重点）
- 第5章 树和二叉树（重点）
- 第6章 图（重点）
- ~~第7章 查找（不考）~~
- ~~第8章 排序（不考）~~

## 大题重点方向

1. **KMP算法** — next / nextval 数组计算
2. **哈夫曼树** — 构建哈夫曼树 + 哈夫曼编码
3. **图遍历** — DFS / BFS 遍历序列
4. **线索二叉树** — 前驱 / 后继查找
5. **邻接矩阵 / 邻接表** — 互相转换
6. **递归转非递归** — 栈的应用
7. **时间 / 空间复杂度分析**
